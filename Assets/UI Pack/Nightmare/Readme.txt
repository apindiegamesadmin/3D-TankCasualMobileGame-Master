UI Pack : Nightmare 1.1.5
------------------------------------------------------------------

	Completed design for toony horror games.

	Features:

		• Backgrounds, particles, alphabets, characters, sounds and more.
		• 336 prefabs.
		• 29 sample layouts.
		• 23 sample UI pages.
		• 17 Sprites.
		• 15 Particles.
		• 4 sounds.
		• 1 demo scene.
		• Customisable and rearrangable.<br>
		• HD textures.

		• Support all build player platforms.
		
	Compatible:

		• Unity 5.5.1 or higher.

	Notes:
	
		1) All sprites and textures are PSD files.
		2) Prefabs are Untiy UI and Unity Sprite.
		3) Buttons, Banners, Gauges, Panel are color-changeable.
		4) Canvases in the demo scene use 0.5 scale factor, so you can use 1.0 scale factor for UHD resolution.
		5) Ground and hills in background are seamless tile-able.
		6) Please feel free to use characters for commercial games.
		7) Fonts are royalty free.
		8) Music by Mike Koenig, Free-Loops.com (http://free-loops.com/7227-synthesizer-heaven.html).
